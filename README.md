# 🌦️ Weather Pipeline Assignment

This project is an automated data pipeline that collects weather data, stores it in a database, generates a poem using an LLM, and publishes the result as a website.

## 🔗 Links

- GitHub Repository:  
  https://github.com/eminagracanin-cloud/weather-pipeline-assignment

- Live Website (GitHub Pages):  
  https://eminagracanin-cloud.github.io/weather-pipeline-assignment/

## ⚙️ Pipeline Overview

The pipeline performs the following steps:

1. Fetches weather data from the Open-Meteo API  
2. Stores data in a SQLite database  
3. Uses Groq (LLM) to generate a bilingual poem  
4. Updates a website with the results  
5. Runs automatically every day using GitHub Actions

## 🚀 Automation

The pipeline runs automatically every day at 20:00 Danish time using a GitHub Actions workflow.
